# datatime - работа с датой и временем (C)

Практическая работа №1  
Вариант 3

---

## Назначение проекта

Проект предназначен для изучения работы со структурами, указателями и динамической
памятью в языке C. Реализована структура `datatime`, представляющая дату и время,
а также набор функций для работы с ней.

Все поля структуры хранятся в динамической памяти. Доступ к данным осуществляется
только через функции, что имитирует объектный подход в языке C.

---

## Структура datatime

typedef struct {
    int *day;
    int *month;
    int *year;
    int *hour;
    int *minute;
} datatime;

Структура может использоваться как для хранения конкретной даты и времени,
так и для представления временного интервала.

---

## Создание и уничтожение объектов

### datatime_create(int d, int m, int y, int h, int min)

Создаёт объект `datatime` в динамической памяти.

Параметры:
- `d` - день
- `m` - месяц
- `y` - год
- `h` - часы
- `min` - минуты

Функция выделяет память под структуру и каждое её поле,
инициализирует значения и возвращает указатель на объект.

---

### datatime_destroy(datatime *dt)

Освобождает всю память, выделенную под объект `datatime`,
включая память под все поля структуры.

---

### create_empty_datatime()

Создание пустого объекта `datatime`,
включая память под все поля структуры.

---

### copy_datatime(datatime *a, const datatime *b)

Копирование первой структуры во вторую

---


## Вывод и сравнение

### datatime_print(const datatime *dt)

Выводит дату и время в формате:

DD/MM/YYYY HH:MM

Функция не изменяет объект.

---

### datatime_compare(const datatime *dt1, const datatime *dt2)

Сравнивает две даты.

Возвращает:
- отрицательное значение - если первая дата меньше второй;
- положительное значение - если первая дата больше второй;
- `0` - если даты равны.

Сравнение выполняется по году, месяцу, дню, часам и минутам.

## Строковый ввод и вывод

### datatime_to_string(const datatime *dt)

Преобразует объект `datatime` в строку формата:

DD/MM/YYYY HH:M

Строка создаётся в динамической памяти.
Освобождение памяти выполняется вызывающим кодом.

Для безопасности используется функция `snprintf`.

---

### datatime_from_string(const char *str)

Создаёт объект `datatime` из строки формата:

DD/MM/YYYY HH:MM

Для разбора строки используется `sscanf`
с проверкой корректности формата.

---

## Работа с временными интервалами

### datatime_to_minutes(const datatime *dt)

Переводит дату и время в общее количество минут.

Используется упрощённая модель времени:
- 1 год = 365 дней
- 1 месяц = 30 дней

---

### datatime_from_minutes(unsigned long long minutes)

Преобразует количество минут обратно в структуру `datatime`.
Полученная структура используется для представления временного интервала.

---

### datatime_print_diff(const datatime *dt1, const datatime *dt2)

Вычисляет временной интервал между двумя датами.
Разница определяется через перевод дат в минуты и обратное преобразование.

---

## Unix time

### datatime_diff_in_seconds_from_unix(const datatime *dt)

Вычисляет количество секунд, прошедших с начала эпохи Unix:

01.01.1970 00:00

Расчёт выполняется с использованием упрощённой календарной модели.

---

## Ввод данных

Пример создания даты через код:

datatime *dt = datatime_create(1, 2, 2026, 14, 30);

Пример создания даты из строки:

datatime *dt = datatime_from_string("01/02/2026 14:30");
